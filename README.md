# jQuery plugin for scrolling in fullscreen sections

## Usage

Returns the most visible Element in viewport that matches a selector.

    $.getMostVisible({
        selector: '.my-container'
    });

## Dependencies
    This plugin makes use of:
    - https://github.com/jquery/jquery
